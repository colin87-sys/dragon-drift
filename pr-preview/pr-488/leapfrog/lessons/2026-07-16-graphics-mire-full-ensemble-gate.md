- **Hierarchy holds:** arch owns the gate frame (gate-clear + nearArchGate visibly working — the gate
  stands in its own water); troughs belong to negative space + the bloom breadcrumb line. Tier discipline
  intact (arch ≫ shroom > bloom in perceived light rank).
