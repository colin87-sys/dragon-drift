# Vesper CP2 — body & tail richness (the "severed tail" defect + the anti-plank body)

**Context.** After CP1 rebuilt the hero wing, CP2 of the rework brought the BODY and TAIL up to the
same richness bar so the wing didn't stand alone on a plain hull. Fable-gated like every checkpoint.
(Backfilled per THE RULE — the CP4 holistic gate caught that CP2 had shipped without a ledger file.)

**What we did.** Deepened the torso stations (a real shoulder swell `ry:0.62` + a haunch mass
`{z:1.10, rx:0.42, ry:0.44, cy:0.18}`), added a dorsal NUB ROW down the spine + neck, haunch flakes,
tail nubs, a mid-tail fin pair, and rebuilt the tail closer as TWIN webbed split fan-fins.

**Lessons banked (each cost a render at the gate):**

1. **A tail fan that isn't WELDED to the stem reads as a "severed tail" — a shipping defect, not a
   nitpick.** The first split fans floated off the stem tip with daylight between them from some chase
   angles. Fix: a CENTER WEB that penetrates the last stem segment forward of the fan roots + rooting
   each fan INTO the stem tip (shared vertices), so there is zero gap from any view. **For any
   multi-part terminal (twin fans, forked tail, split plume): build a connective web first, then hang
   the parts off it — never place the parts and hope they meet.**

2. **A haunch you modelled in the profile can still be INVISIBLE in silhouette.** Deepening the loft
   station wasn't enough — from rear-chase the haunch mass didn't read. It needed an explicit dorsal
   BUMP (a raised facet on the hip) to break the outline. **Silhouette richness is about the OUTLINE,
   not the cross-section: if a mass doesn't change the rear-chase profile, the player never sees it.**

3. **A spine ridge must physically REACH the skull or it reads as stopping mid-back.** The dorsal nub
   file ran out over the shoulders; the gap between the last nub and the head read as "unfinished."
   Fix: continue the nub row up the NECK stations to the skull base. **A repeating detail row has to
   run all the way to its anatomical terminus — a row that fades out mid-run looks like a bug.**

4. **Echo the hero for cohesion.** The split fans were rebuilt from flat petals into MINI fingered-
   webbed fans (radiating bones + inward-cupped membrane bays, the CP1 wing recipe in miniature) so
   the tail speaks the same language as the wing. A premium creature reads as one design when its
   landmarks rhyme.

**→ Unlocks.** A body/tail that carries its own richness under the hero wing: the welded-web terminal
pattern, the silhouette-bump rule, the reach-the-terminus rule, and the hero-echo cohesion move — all
reusable for any creature whose extremities must read as finished, attached, and of-a-piece. Next was
CP3 (motion) and CP4 (the 4th value tier + the holistic gate).
