# Fresh-Five premium sheets — harsh rear-chase gate pass

**What we did.** Ran the Fable rear-chase critic protocol at max rigor (on Opus, since Fable is
rate-limited) against the five premium build sheets — Tempest, Stiletto, Sylph, Revenant, Tocsin.
Judged each as a black-fill seen from BEHIND (95% of play), then judged the five as a SET. Edited all
five sheets + `FRESH-DRAGONS-SYNTHESIS.md` with a sharpened rear-chase silhouette statement, a
buildability-audit table (every hero element + motif → a cited engine construction path + overdraw
cost), and an SSSR appeal line. Verdicts: Revenant PASS; Tempest / Stiletto / Sylph / Tocsin REVISE.

**What we learned / the gotchas.**
- **A dorsal-mounted sheet is EDGE-ON from dead astern.** Sylph's "aurora curtain" was a near-VETO:
  a vertical dorsal veil collapses to a thin line in rear black-fill (the #1 flat-plane/kite failure).
  Fix without a concept swap: FLARE it outboard into a hood (±10°→±35° pleat cant, width ≥1.1× girth)
  and promote the streamers to span-carriers. Rule: a rear-chase wing must occupy WIDTH across the
  view, not lie in the sagittal plane.
- **N equal horizontal repeats = venetian-blind read.** Tempest's 3 stacked decks needed a ×0.8
  chord-decay + set-back so the stack reads as a tapering anvil cone, not parallel slats. Assert "no
  two decks within 10% chord."
- **The closest set-collision was Tempest ↔ Tocsin** — both broad wings with a ~5-element bottom
  fringe. Separated on attitude (high-up vs low-flat), inner grain (stripe-slabs vs round coins), and
  fringe character (wispy tapering virga vs rigid straight rods). Broad-wing + bottom-fringe is a
  recurring near-collision shape; separate it on those three axes.
- **Transiently-lit / additive motifs don't carry the silhouette.** Tocsin's rings fade at distance and
  Tempest's strike is 6–18% duty — so the STANDING (unlit) frame must already be cool. Never let a
  spectacle be the crutch for a weak standing outline.

**Reusable pattern.** The buildability audit table is the fastest veto tool: force every hero element +
motif to name its reference implementation (`knapLoft`, Vesper `seam` mat, `dragonSideFins.buildSail`,
`shingleRow`, `TorusGeometry` per shipped `js/rings.js`, `pulseTimer.js`) and its overdraw cost. Anything
that can't cite a path is vetoed. `js/rings.js` (TorusGeometry + CPU scale + intensity fade) is the
proven precedent for any expanding-shockwave motif.

**What it unlocks.** The five sheets now pass the three owner directives (rear-chase silhouette /
buildability / SSSR appeal) subject to the four REVISE fixes being built. Residuals are motion/feel only
(ripple rate, TOLL rhythm, four-wing shimmer, SNAP beat) — judged by the human on the live PR preview.
