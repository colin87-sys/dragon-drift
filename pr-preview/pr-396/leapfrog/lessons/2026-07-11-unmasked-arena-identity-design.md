# UNMASKED arena identity — design the MEANING into channels the plumbing already has

**What we did.** Wrote `docs/unmasked-arena-identity.md` — the creative identity for the finale's
two mid-fight arenas (S2 **THE HOLLOW BEHIND THE SKY** — "Where It Made the Masks"; S3 **THE
UNVEILED HEAVEN** — "What the Sky Was a Mask Of") on top of the transformation plan's value-space
env override + its audit's rails. No code. Decided O6 (void dressed with THE BLANKS — six
unfinished mask-ovals, one cracked = Voidmaw's, empty-first ship order) and authored the heaven
palette at the audit's real ceiling.

**What we learned / the reusable patterns:**

- **Re-derive palette hexes with the gate's own `lum()` BEFORE writing them into a design doc.**
  The plan's heaven (L≈0.88) was already proven unshippable by the audit (amber/cyan have no
  override lever); a 5-minute node one-liner using `bulletcontrast.mjs`'s exact formula found a
  gold-forward set at fog .732 / horizon .744 where all eight role colours pass via the revived
  layered read. An identity doc whose hexes fail the shipped gate is fiction — run the numbers in
  the design session, not the build session.
- **The strongest "another dimension" move is a MEANING flip on an existing render, not a new
  render.** `starMix → 1` is one uniform — but naming the void's stars as *pinholes in the mask*
  retcons the entire game's night sky for free, and gives the S2→S3 gold flood its causality (the
  light was always behind the dark). Look for the cheapest channel that can carry the heaviest
  lore before authoring any new form.
- **Dress ONE arena, empty the OTHER — opposition beats decoration.** The void gets the only
  authored forms (the blanks = the workshop); the heaven is deliberately form-free (the court —
  the boss's own halo/starburst IS the architecture). Two arenas that both get props read as two
  biomes; furniture-vs-light reads as a transformation.
- **New silhouette-forms in a value-space feature must carry their own ship order.** The blanks
  are the one thing the env override can't do (a real mesh group) — so the identity ships
  empty-first (PR-A unchanged) and the forms land as their own def-gated increment with their own
  §2 probe + Fable gate. Never make a hero read hostage to the only new geometry in the design.
- **Anti-scope is part of an identity spec.** Rejected dressings are written down WITH reasons
  (leash-chains answer the post-game gap early; sky-ribs read as MARROWCOIL) so later sessions
  don't re-add them in a polish pass.

**What it unlocks.** The harsh-critic audit has concrete claims to attack (every hex has a
measured L; every form has a draw/overdraw budget; every lore rhyme cites its web thread), and
PR-A/B/C can build to a named identity instead of placeholder palettes. Owner decisions
outstanding: blanks sign-off, the cracked blank's identity, arena sub-lines on stage cards, the
"stars are holes" rider line, fever policy.
