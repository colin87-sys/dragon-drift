# Style exploration boards — Gate 1

Five candidate design languages for the new dragon roster, all rendering the **same
subject** (a mid-tier dragon: deep chest, slimmer hips, five-finger membrane wings,
whip tail, horned head on an S-curved neck) so the comparison isolates *style*, not
creature. Each board: ¾ hero · side ortho · chase-cam view · blackout silhouette.

These images are **direction only** (DRAGON-DESIGN-SYSTEM.md §1.2) — the pick decides
the curve language, finish, and palette philosophy the procedural builds aim for; it is
never a match target.

| Board | Language | In renderer-lever terms |
|-------|----------|------------------------|
| `board-1-apex.png` | Sleek apex predator (Toothless lineage) | smooth continuous hull loft, low `scaleRelief`, matte finish (`bodyMetalness` 0, high roughness), big `eyeScale`, rounded wing scallops, gentle `cy` arch |
| `board-2-western.png` | Majestic regal western | heavy chest stations (D1 pushed high), horn-crown layers, broad `rootChord` cathedral wings, warm two-tone palette (dominant hide + pale belly) |
| `board-3-celestial.png` | Elegant celestial/ethereal | elongated station layout, strong S-curve `cy`, ribbon `surfaceLayers` fins, emissive accent seams (HDR-overdriven, `toneMapped=false`), maximal S2 negative space |
| `board-4-mythic.png` | Sculpted mythic idol (the boss-arc language) | painted value tiers across 2–3 material groups, gilt relief bands (authored, mirrored — L126), glow seams between plates, hard-edged outline points |
| `board-5-aero.png` | Streamlined aero speed-form | swept delta `wingForms` (raked tips), faceted section profile (`dragonFaceted` lineage), energy-seam accents, everything raked toward −z |

**Gate 1 verdict (human, 2026-07-03):** tier-mapped — **rounded apex curves** (board 1)
for the cute free starter · **western/celestial** (boards 2+3) for the mid tier ·
**sleek mythic** (board 4 on a sleeker curve language) for the SSSR premium. Details in
DRAGON-DESIGN-SYSTEM.md §3.
